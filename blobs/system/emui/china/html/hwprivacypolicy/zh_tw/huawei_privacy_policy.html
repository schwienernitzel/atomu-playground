一、華為會如何收集和使用您的個人資訊

（一）華為收集的個人資訊

您在使用華為產品或服務時，可能需要提供個人資訊。您不一定要向華為提供個人資訊，但在某些情況下，如果您選擇不提供，華為將無法為您提供相關產品或服務，也無法回應或解決您所遇到的問題。

華為僅會出於本聲明所述目的收集和使用您的個人資訊。下文舉例說明了我們可能收集的個人資訊：

1. 您向華為提供的資訊

您需要註冊華為 ID 才能享受某些功能或服務。當您註冊華為 ID，或者登入華為 ID 於線上購物、下載軟體或購買服務時，我們會要求您提供相應的個人資訊，例如您的姓名、郵件地址、手機號碼、訂單資訊、收貨地址、付款方式等。

華為的某些產品可讓您與他人通訊並分享資訊。當你使用華為產品與家人和朋友分享內容時，您可能需要建立公開顯示的華為 ID 個人檔案，包括暱稱、頭像等。華為可能會收集您提供的與前述人士有關的資訊，例如姓名、電子郵件地址及電話號碼等。華為將採取合理且必要的措施來保障前述通訊的安全性。

為滿足部分司法管轄區對帳號實名制、遊戲防沉迷系統或網際網路支付的要求，華為可能會要求您提供由政府發放的身分證明或能表明您身分的相關卡證資訊。

2. 華為在您使用服務過程中獲取的資訊

華為會收集您的裝置資訊，以及您和您的裝置如何與華為產品與服務交互的資訊，此類資訊包括：

(1) 裝置及應用程式資訊。例如裝置名稱、裝置識別碼 (IMEI、ESN、MEID 和 SN)、裝置啟用時間、硬體型號、作業系統版本、應用程式版本、軟體識別碼、裝置及應用程式設定 (如地區/語言/時區/字型) 等。

(2) 行動網路資訊。例如公共陸地行動網路提供商 ID (PLMN)、地理位置 (裝置定位所在區域訊號發射塔識別資訊)、網際網路協定 (IP) 位址。

(3) 記錄資訊。當您使用華為的服務或者查看由華為提供的內容時，華為會自動收集某些資訊並儲存在記錄中，例如服務的存取時間、存取次數、存取 IP、事件資訊 (如錯誤、當機、重新開機、升級) 等。

(4) 位置資訊。當您存取某些基於位置的服務時 (例如執行搜尋、使用導航軟體或查看某個位置的天氣時)，華為會收集、使用並處理您裝置的近似位置或精確位置，這些位置資訊透過 GPS、WLAN 和服務提供商的網路 ID 獲取。華為會詢問您要為哪些應用程式啟用基於位置的服務。您可在裝置的設定功能表中選擇關閉裝置上的相應權限，以拒絕共用您的位置資訊。

(5) 您儲存在華為伺服器中的內容。例如，您上傳至雲端儲存空間的內容會儲存在華為伺服器中，方便裝置之間進行快速存取和資訊共用。華為不會查看您儲存在華為伺服器中的內容。

3. 第三方來源資訊

在法律允許的情況下，華為還會從公用和商用來源獲取您的相關資訊。我們還會從第三方社交網路服務獲取您的相關資訊，例如您何時使用社交網站帳戶登入我們的網站。

華為也可能透過其他方式收集有關您、您的裝置或服務使用情況的其他資訊，這些方式會在收集時向您說明，或者征得您的同意。

4. 征得授權同意的例外情況

在以下情形中，華為收集和使用個人資訊前無需獲得您的授權同意：

(1) 與國家安全、國防安全有關的；

(2) 與公共安全、公共衛生、重大公共利益有關的；

(3) 與犯罪偵查、起訴、審判和判決執行等有關的；

(4) 為了維護個人資訊主體或其他個人的生命、財產等重大合法權益但又很難得到本人同意的；

(5) 所收集的個人資訊是個人資訊主體自行向社會公眾公開的；

(6) 從合法公開披露的資訊中收集您的個人資訊的，如合法的新聞報導、政府資訊公開等管道；

(7) 根據您的要求簽訂合約所必需的；

(8) 用於維護所提供產品與/或服務的安全穩定運作所必需的，例如發現、處理產品與/或服務的故障；

(9) 為合法的新聞報導所必需的；

(10) 學術研究機構基於公共利益開展統計或學術研究所必要，且對外提供學術研究或描述的結果時，對結果中所包含的個人資訊進行去標識化處理的；

(11) 法律法規規定的其他情形。

5. 非識別性資訊的收集和使用

非識別性資訊是指無法用於確定個人身分的資訊，例如網站瀏覽量、應用程式下載量、產品銷量等。華為會收集匯總的統計資訊，目的在於瞭解使用者如何使用我們的產品與服務。藉此，華為可以改善自己的服務，更好地滿足使用者的需求。華為可能會自行決定出於其他目的收集、使用、披露或轉移非識別性資訊。

（二）華為會如何使用您的個人資訊

華為會將您的個人資訊用於以下目的：

1. 註冊、啟用為您購買的華為個人及家用產品；

2. 註冊華為 ID，以便您享受更豐富多彩的功能；

3. 交付、啟用或驗證您所請求的產品與服務，或應您的要求對前述產品與服務進行變更、提供技術支援與售後服務；

4. 向您傳送作業系統或應用程式更新和安裝的通知；

5. 為您提供個人化使用者經驗和個人化內容；

6. 在您明確同意或應您主動要求的情況下，與您聯絡、向您傳送有關您可能感興趣之產品與服務的資訊、邀請您參與華為促銷活動和市場調查，或向您傳送行銷資訊。如果您不想接收此類資訊，可以隨時退訂；

9. 同步處理、共用和儲存您上傳或下載的資料以及執行上傳和下載所需的資料；

10. 改善我們的防損和反欺詐計劃；

11. 其他征得您同意的目的。


二、華為會如何使用 Cookie 和同類技術

華為的網站、行動應用程式軟體、線上服務、電子郵件及廣告可能會使用 Cookie、像素標籤和網站指標等本機儲存技術。華為將透過 Cookie 和同類技術收集的資訊視為非個人資訊。但是，如果當地法律將 IP 位址或類似的識別標記視為個人資訊，則華為亦將此等識別標記視為個人資訊。

（一）Cookie

Cookie 是一種由網路伺服器儲存在電腦或行動裝置上的純文字檔。Cookie 的內容只能由建立它的伺服器擷取或讀取。每個 Cookie 對您的網路瀏覽器或行動應用程式都是唯一的。Cookie 通常包含識別碼、站點名稱以及一些號碼和字元。

華為有時會在電腦或行動裝置上儲存 Cookie，我們啟用 Cookie 的目的在於改善使用者經驗，包括：

1. 登入和驗證。當您使用華為 ID 登入網站時，藉由 Cookie，您可以直接從網站的一個頁面跳轉到另一個頁面，而不必在每個頁面上重新登入；

2. 儲存您的喜好設定和其他設定。網站能夠藉由 Cookie 來儲存設定，例如電腦或行動裝置的語言、文字大小、購物車及其他瀏覽喜好設定等；

3. 基於興趣的廣告。華為使用 Cookie 來收集有關您的線上活動的資訊並發現您的興趣所在，同時向您提供與您相關度最高的廣告。

4. 統計分析。藉由 Cookie，華為可以收集有關您使用我們網站和其他應用程式的資訊，例如記錄使用者的單次存取 (使用工作階段 Cookie) 或多次存取 (使用永久 Cookie)。

華為不會將 Cookie 用於本聲明所述目的之外的任何其他用途。您可以根據自己的喜好來管理或刪除 Cookie。如需詳細資料，請參閱 AboutCookies.org。您可以清除電腦上儲存的所有 Cookie，大部分網路瀏覽器都設有封鎖 Cookie 的功能。如需詳細瞭解如何變更瀏覽器設定，請存取以下連結：<Internet Explorer>、 <Google Chrome>、<Mozilla Firefox>、<Safari> 和 <Opera>。

如果您清除 Cookie，則需要在每一次造訪華為的網站時親自變更使用者設定。同時也請注意，華為的某些服務可能必須使用 Cookie，停用 Cookie 可能會影響您使用這些服務的全部或部分功能。

（二）網站指標/像素標籤

除 Cookie 外，華為及某些第三方還會在網站上使用網站指標和像素標籤。網站指標通常是一種嵌入到網站或電子郵件中的電子影像，可用於在您查看網頁或電子郵件時識別您裝置上的 Cookie。像素標籤能讓華為以您可閱讀的格式傳送電子郵件，並讓我們獲知電子郵件是否被開啟。

華為以及某些第三方將該類技術用於多種用途，包括分析服務使用情況 (與 Cookie 配合使用) 以及提供更符合您需求的內容和廣告。例如，華為向您傳送的電子郵件可能含有連結至華為網站內容的輕觸 URL。如果您輕觸該連結，華為則會跟蹤此次輕觸，協助我們瞭解您的產品與服務喜好並改善客戶服務。如果您不希望自己的活動以這種方式遭到追蹤，您可以隨時從華為的寄信名單中退訂該服務。

（三）其他本機儲存

華為以及某些第三方可能會在某些產品與服務中使用其他本機儲存技術，例如本機共用物件 (又稱「Flash Cookie」) 和 HTML5 本機儲存。這些技術與上述 Cookie 類似，它們同樣儲存在您的裝置中，並且可用於記錄有關您的活動和喜好的特定資訊。但這些技術與標準 Cookie 所使用的裝置可能不同，因此，您可能無法使用標準瀏覽器工具和設定來控制它們。如需有關停用或刪除 Flash Cookie 中所含資訊的說明，請按一下此處。

（四）Do Not Track (請勿追蹤)

許多網路瀏覽器均設有「請勿追蹤」功能，該功能可向網站發佈「請勿追蹤」請求。目前，主要的網際網路標準化組織尚未設立相關政策來規定網站應如何應對此類請求。如果您的瀏覽器啟用了「請勿追蹤」功能，那麼華為的所有網站都會尊重您的選擇。


三、華為如何委託處理、共用、轉讓、公開披露您的個人資訊

（一）委託處理

在某些情況下，華為會委託其他公司代表華為處理您的個人資訊。例如代表華為處理熱線、傳送電子郵件、提供技術支援的公司，這些公司只能將您的個人資訊用於代表華為向您提供服務。

華為會與被委託方簽署嚴格的保密合約或個人資訊處理條款，要求他們按照本聲明並採取相關的保密和安全性措施來處理個人資訊。

（二）共用

共用是指華為向其他個人資訊控制者提供個人資訊，且雙方分別對個人資訊擁有獨立控制權的過程。華為不會對外共用您的個人資訊，但以下情況除外：

1. 在獲取明確同意情況下的共用：獲得您的明確同意後，華為會向您指定的第三方共用您授權範圍內的資訊；

2. 在法定情形下的共用：華為可能會根據法律法規規定、訴訟爭議解決需要，或按行政、司法機關依法提出的要求，對外共用您的個人資訊；

3. 共用給華為的附屬公司：您的資訊可能會在華為的附屬公司內共用。我們僅會出於特定、明確而合法的目的在華為的附屬公司內共用您的資訊，並且只會共用提供服務所必要的資訊。例如，在註冊華為 ID 時為了避免重複註冊，我們需要對擬註冊的 ID 進行全球唯一性驗證。

4. 共用給業務合作夥伴：華為可能會向合作夥伴等第三方共用您的訂單資訊、帳戶資訊、裝置資訊及位置資訊，以確保為您提供的服務順利完成。但我們僅會出於合法、正當、必要、特定、明確的目的共用您的個人資訊，並且只會共用提供服務所必要的個人資訊。我們的合作夥伴包括：

(1) 第三方賣家和第三方開發者：某些產品或服務由第三方直接向您提供，華為須將交易相關資訊共用給第三方，來滿足您向其購買商品或服務的需求。例如，您在應用程式中心購買其他開發者的商品時，我們須與該開發者共用您的必要資訊，交易才能完成。

(2) 商品或技術服務的供應商。華為可能會將您的個人資訊共用給支援我們功能的第三方，包括為我們供貨或提供基礎架構技術服務、物流配送服務、支付服務、資料處理服務的第三方等。我們共用這些資訊的目的是為了實現產品及服務的功能，例如，我們必須與物流服務提供商共用您的訂單資訊，以安排送貨；或者我們需要與第三方支付機構共用您的訂單號和訂單金額，以讓其確認您的支付指令並讓您完成支付等。

在 3-4 下，華為會與共用個人資訊的公司、組織和個人簽署嚴格的保密合約，要求他們按照本聲明並採取相關的保密和安全性措施來處理個人資訊。

（三）轉讓

轉讓是指將個人資訊控制權由華為向另一個個人資訊控制者轉移的過程。華為不會將您的個人資訊轉讓給任何公司、組織和個人，但以下情況除外：

1. 在獲取明確同意的情況下轉讓：獲得您的明確同意後，我們會向其他方轉讓您的個人資訊；

2. 在涉及合併、收購或破產清算時，如果涉及到個人資訊轉讓，我們會要求持有您個人資訊的新公司或組織繼續受本聲明的約束，否則我們將要求該公司或組織重新向您徵求授權同意。

（四）公開披露

華為僅會在以下情況下，公開披露您的個人資訊：

1. 獲得您的明確同意後；

2. 基於法律或合理依據的公開披露：在法律、法律程序、訴訟或公共和政府主管部門所要求的情況下，華為可能會公開披露您的資訊；

（五）共用、轉讓、公開披露個人資訊時事先征得授權同意的例外情況

在以下情形中，共用、轉讓、公開披露您的個人資訊時無需事先征得您的授權同意：

五、您如何管理您的個人資訊

（一）存取、更正、刪除

在華為提供產品及服務的眾多國家或地區中，部分國家或地區的立法規定了個人資訊主體有權存取、更正、刪除華為所保留的相關個人資訊等權利。在前述國家或地區，個人資訊主體或其代理人可以向華為提出存取、更正、刪除華為所保留之相關個人資訊的請求 (下文統稱「請求」)。

1. 提出請求的方式與管道

個人資訊主體請求必須以書面形式提出。即便請求者未說明請求所依據的法規，請求同樣有效。一般來說，口頭提出的存取請求是無效的，若一些國家的立法亦承認口頭請求的有效性，華為將會遵循此類規定。

個人資訊主體請求可以透過華為消費者業務官方網站、HiCare 應用程式、華為帳號應用程式提出。如果個人資訊主體透過熱線、服務信箱、線上客戶服務、服務門店等其他管道提出相關請求，我們會引導個人資訊主體透過前述管道提出正式訴求，以便於交流和反饋處理進度及結果。華為建立專門的個人資訊主體請求管道，旨在保護個人資訊主體的合法利益，保障華為的正常營運，並防止個人資訊請求權利被他人盜用或濫用。

2. 請求的有效性

大多數的法律要求個人資訊主體提出請求的時候遵循特定的要求。本聲明要求個人資訊主體做到以下幾點：

(1) 透過華為專門的個人資訊主體請求管道 (即華為消費者業務官方網站、HiCare 應用程式、華為帳號應用程式) 提出請求；

(2) 提供足夠的資訊給華為來驗證請求者的身分 (確保提出請求的人是個人資訊主體本人或其授權者)；

(3) 請求內容具體，具有可執行性。

3. 請求的處理期限

華為會竭盡全力確保在個人資訊主體提出存取請求後的一個月內給予回覆，考慮到請求的複雜性和數目，在必要的時候，這一期限可能再延長兩個月。對於延期提供資訊的情況，華為將通知個人資訊主體相關情形和延遲原因。如果本條所設定的時限與當地法律規定發生衝突，則以當地法律規定為准。

4. 請求的結果

個人資訊主體提出請求後，將可能出現以下幾種結果：

(1) 請求被駁回

在某些情況下，個人資訊主體的請求將被駁回，包括但不限於：

a. 當地法律未賦予個人資訊主體享有相關權利時；

b. 無法驗證提出請求之人的身分時；

c. 個人資訊主體提出的請求無法驗證或超出範圍，尤其是重複提出請求時；

d. 如果涉及到的資訊跟華為在糾紛中要做出的賠償或獲得的補償有關，披露資訊很有可能會損害華為的利益時；

e. 如果保留資訊只是出於統計和研究目的，而且統計和研究結果的公佈不會披露個人身分時；

f. 法律規定的其他情形。

如果華為駁回了個人資訊主體的存取請求，將正式向請求者說明原因。

(2) 請求成功

當 (1) 中的情形未發生時，華為會滿足個人資訊主體的請求。為了提高請求成功的可能性，請在提出請求時盡可能提供詳細的資訊，例如請求的類別和具體內容、資訊持有者的相關資訊 (例如您使用的華為產品和服務名稱)、資訊產生或處理的時間範圍 (時間範圍越小，成功的可能性越大) 等。

5. 特別聲明

(1) 任何人均無權存取其他人的個人資訊，除非是被授權者或監護人。

(2) 大多數的法律規定都明確了組織可以不向個人資訊主體提供資料的情況。這些情況包括以下幾種：提供了資料會破壞打擊恐怖主義的力量、個人資訊主體已經重複申請了多次，或者獲取和提供資訊會耗費不相稱的資源等。

(3) 原則上，華為不會提供以下資訊：

a. 關於其他人的資訊 – 個人資訊主體在存取請求中想要獲取的資料可能會涉及到個人資訊主體之外的其他人。除非所涉及的個人同意，否則華為不會提供此類資訊。

b. 重複申請 – 請求者提出的是針對同一個個人資訊主體的相同或類似請求，資料在過去一段時間內沒有發生變更，而且華為已經提供了資料，在這種情況下，華為通常不會再提供資料的副本。此外，華為也沒有義務來提供已經公諸於眾的資訊。

c. 保密下給出的觀點 – 如果關於個人資訊主體的資訊是在保密下給出的觀點，華為沒有義務來提供這些資訊。

d. 特殊檔案 – 華為保留的任何特殊資訊是不會在回覆個人資訊主體的存取請求中予以公開的。一般而言，特殊資訊包括任何機密檔案 (例如客戶與其律師之間的直接溝通)，以及為取得或給出相應法律意見而保留的資訊 (不論是訴訟本身還是法院訴訟過程中所涉及到的資訊)。
